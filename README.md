# odin-tic-tac-toe

objects for game, player, gameboard.


Play Game Function:
-Start Game
create two players
assign them tokens
create board

-Play Round
select which player starts
update board with new block with player token

-Check Winner
check if winning conditions met
if not:
switch which players turn
go to Play Round function

-Print Winner

-Reset Game
clear board
clear players
